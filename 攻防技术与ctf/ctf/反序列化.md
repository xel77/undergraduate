# **反序列化**

两种类型：

1. php的反序列化
2. java的反序列化

介绍：

​	反序列化，它发生在应用程序将序列化的数据（通常是从不可信的源接收的数据）反序列化成对象时。序列化是将对象状态转换为可以存储或传输的格式的过程，而反序列化则是相反的过程，即将这些格式转换回对象。

简单点讲就是就接受到的数据转换成可用格式一般为json的过程叫做序列化，逆向过程就是反序列化。

- php的反序列化

  ```php
  <?php 
      class person{
      public $name;
      public $age;
      public $sex;
      public function say(){
          echo "my name is ".this->$name.".";
      }
  }
      
      
      ?>
  ```
